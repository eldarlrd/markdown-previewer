## Install
```sh
pnpm i
```
## Develop
### Run Env.
```sh
pnpm dev
```
### Prod. Build
```sh
pnpm build
```
### Preview Prod. Build
```sh
pnpm preview
```
### Format Code (ESLint + Stylelint + Prettier)
```sh
pnpm format
```
## Deploy
```sh
pnpm deployer
```
